# GooglePayCardInfo

  Unencrypted card metadata from
  [CardInfo](https://developers.google.com/pay/api/web/reference/response-objects#CardInfo).

## Example Usage

```typescript
import { GooglePayCardInfo } from "@moovio/sdk/models/components";

let value: GooglePayCardInfo = {
  cardNetwork: "VISA",
  cardDetails: "1234",
  cardFundingSource: "DEBIT",
  billingAddress: {
    countryCode: "US",
  },
};
```
